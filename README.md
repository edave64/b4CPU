# b4CPU (b4cpu)

A small CPU you can follow

## Install the dependencies
```bash
npm install
```

### Start the app in development mode (hot-code reloading, error reporting, etc.)
```bash
npm run dev
```


### Lint the files
```bash
npm run lint
```


### Format the files
```bash
npm run format
```


### Build the app for production
```bash
npm run build
```
